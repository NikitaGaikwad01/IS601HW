Linux Command

ls – directory listing
ls -al – formatted listing with hidden files
cd dir - change directory to dir
cd – change to home
pwd – show current directory
mkdir dir – create a directory

Vi Command

I = insert mode
Escape = get out of insert mode
Shift key + colon = command prompt for vi
w = write
q = quit
wq = write and quit

Git Command

git init = create the repository
git add filename = add the files to git in preparation for a commit
git commit -m "message" = do the commit to save changes
git checkout -b branch name = checkout branch
git checkout branch name = checkout branch
git merge branchname = merge the branch
Note: You have to be on the branch that you want to merge to
